# 📜 ისტორიის გამოცდის ქვიზი

ინტერაქტიული ქვიზის საიტი ისტორიის გამოცდისთვის მოსამზადებლად. ყველა კითხვა და პასუხი ქართულ ენაზე.

## ფუნქციები

- 17 თავი ანტიკური და ადრე-შუა საუკუნეების ისტორიიდან
- მრავალპასუხიანი ტესტი ყოველი თავის ბოლოს
- მყისიერი უკუკავშირი (სწორი/არასწორი)
- დეტალური ახსნა-განმარტებები
- ქულის თვალყურის დევნება
- "ყველა თავი ერთად" რეჟიმი — შერეული კითხვები
- კლავიატურის მართვა (1-4 პასუხებისთვის, Enter შემდეგისთვის)
- რესპონსიული დიზაინი (მუშაობს ტელეფონზე და დესკტოპზე)
